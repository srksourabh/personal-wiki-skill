## Key Learnings & Best Practices

### Campaign Structure
✅ 15-day repeating cycle optimal for consistency
✅ 4 platforms × daily posting (9:30, 11:30, 14:30, 16:30 IST)
✅ Platform-specific messaging (LinkedIn ≠ Instagram ≠ X tone)
✅ Daily themes drive coherent 15-day narrative

### Content Strategy
✅ Product positioning → daily messaging angles
✅ 7-8 thematic hooks across 15 days (social proof, pain point, differentiation, urgency, etc.)
✅ No copy-paste: each post is unique
✅ CTA consistency (website link in every post)

### BYOK Model
✅ Users bring their own API keys (Gemini, OpenAI)
✅ No credential persistence (no privacy leaks)
✅ Cost control (users pay for their own API consumption)
✅ Graceful fallback to deterministic JS if LLM unavailable

### Autonomous Scheduling
✅ CSV export → Postiz import (1-click deployment)
✅ No manual queue, fully automated publishing
✅ 15-day cycle repeats indefinitely
✅ Zero operational overhead post-deployment
